# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.x     | ✅        |

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Email `security@pamoja.app` with:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes

We will acknowledge within 48 hours and aim to resolve critical issues within 7 days.

## Scope

- Soroban smart contracts (`contracts/`)
- Frontend application (`frontend/`)
- Wallet integration and key handling

## Out of Scope

- Third-party dependencies (report to their maintainers)
- Stellar network-level issues (report to SDF)
